🚀 KeyMod BETA v0.1 Features, Plugins, and Purpose

🌐 Mod Purpose

KeyMod is a full RocketGoal enhancement suite built to:

Improve live gameplay visibility (score HUD, shot analysis, streaks),

Add streamer-safe overlays (name/MMR blur, screenshot/clip tools),

Provide performance & quality tweaks (optimizer, visual FX),

Add persistent next-gen stats and match memory,

Offer a plugin-based modular UI for customization.

🔧 Core Runtime Features

F2/backtick/1 menu toggle with draggable/resizable KeyMod UI.

Top-left live score board with current mode, score, opponent + MMR, 1v1/2v2/3v3-awareness.

Shot Analysis HUD with goals/shots/accuracy and post-match insight.

macOS-style loading screen locked to the update style.

Toast notifications with permanent, full-width draggable style.

Goal performance lessons and daily streak counters.
